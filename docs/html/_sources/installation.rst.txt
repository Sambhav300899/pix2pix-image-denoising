Installation
================
To install the library follow the instructions below.

Clone the library
--------------------------------
.. code-block:: bash

    git clone https://github.com/Sambhav300899/pix2pix-image-denoising.git

Installing library in dev mode
--------------------------------
.. code-block:: bash

    pip install -e ".[dev]"

Installing library in normal mode
----------------------------------
.. code-block:: bash

    pip install .
